## 📝 Adding New Documents

1. Create folder for new app: `mkdir {app-name}`
2. Add HTML files: `privacy-policy.html`, `terms-of-service.html`
3. Update `index.html` with new app section
4. Commit and push to main branch
5. GitHub Pages automatically deploys (usually within 1-2 minutes)
